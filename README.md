# Reservation App

Written in React. Uses [Primereact](https://primereact.org/) and
[Primeflex](https://primeflex.org/).

## Setup

```
npm install
```

## Dev

```
npm run dev
```

Before pushing to the GH repo, please run. Please, do not push to `master`.

```
npm run lint:fix
```
